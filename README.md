# Hugs & Chocolates Confetti App

A polished single-page celebration app where users can tap **Send Hugs** or **Send Chocolates** and trigger themed emoji-confetti bursts.

## Features

- Two themed celebration actions:
  - 🤗 Hug burst
  - 🍫 Chocolate burst
- Emoji confetti system with randomized spread, travel distance, and rotation.
- Live counters for total, hugs, and chocolates bursts.
- Activity feed with timestamps and quick clear action.
- Responsive, mobile-friendly layout.

## Run locally

Because this is a static app, run it with any local server:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
